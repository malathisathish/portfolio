# Malathi Y - Professional Portfolio v2

This is a modern, professional portfolio built with React and Vite, featuring a premium glassmorphism design tailored for a Data Scientist.

## How to Run Locally

1.  **Navigate to the project directory**:
    ```bash
    cd portfolio-v2
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```

4.  **View in Browser**:
    Open [http://localhost:5173](http://localhost:5173) in your web browser.

## Features
- **Modern Design**: Dark mode with glassmorphism effects.
- **Career Narrative**: Highlights transition from Nursing to Data Science.
- **Projects Showcase**: Interactive cards for technical projects.
- **Responsive**: Works on desktop, tablet, and mobile.
